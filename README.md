# Learing-Docker

Learn Docker 🐳

This repository is a complete learning journey for Docker, covering the fundamentals of containerization and practical Docker usage.

The goal of this repository is to understand how Docker works, how to create and manage containers, build custom images, configure environments, and use Docker in real-world software development workflows.

Topics Covered
Introduction to Docker and containerization
Docker architecture and core concepts
Installing and configuring Docker
Working with Docker images and containers
Writing Dockerfiles
Building custom Docker images
Container networking
Docker volumes and persistent storage
Environment variables and configuration management
Docker Compose for multi-container applications
Deploying applications using Docker
Docker best practices
Technologies
Docker
Docker Compose
Linux
Node.js applications
Container-based development workflows
